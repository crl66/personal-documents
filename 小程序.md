### input组件有时在ios键盘不唤醒

```
步骤一：若已在 input 中设置了 enableNative 属性，删除 enableNative 属性的全部设置。
步骤二：在 app.json 文件 window 对象内，设置 "enableInPageRenderInput":"YES"。
```
